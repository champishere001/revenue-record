e-SIGMA क्या है?

Full Form

e-Survey Instrument and Generalised Multimodal Application

Cloud-based platform for NSS/NSO surveys (PLFS, ASI, ASUSE, CAPEX)

CAPI (Computer Assisted Personal Interviewing) + Peripheral Modules

Paperless data collection with real-time sync

Faster Publication: 90-120 days for annual surveys

क्यों महत्वपूर्ण है?

पहले (Before)

  • Manual task allocation
  • Delayed data scrutiny
  • Unequal workload distribution
  • Data quality issues

e-SIGMA Solution

  • Centralized control
  • Role-based access control
  • Real-time progress tracking
  • Timely, reliable statistics

Impact: Better policy-making through accurate, timely data
