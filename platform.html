1. Defining Platform Work

1

The Conceptual Framework

Platform work is an arrangement where matching between workers and clients is intermediated through a digital or online platform.

Location-Based Services

  • On-demand work via app-based platforms.
  • Requires physical presence at delivery.
  • Examples: Uber/Ola, Zomato, Healthcare.

Web-Based Freelancing

  • Remote work, digitally mediated.
  • Higher-skilled professional services.
  • Examples: Graphic design, Coding, Legal.

2. Qualifying as a Platform

2

Beyond Simple Listing

The platform must actively organize key aspects of the activity:

  • Active Matching: Significant role in connecting customer and worker (not just a directory).
  • Work Organization: Assigning nearest worker or delivery routes.
  • Exerting Control: Price setting, rules of conduct, performance monitoring (ratings), and restrictions on flexibility.
Crucial Logic: If a customer contacts a provider directly using information found on a website (like a phone directory), it is NOT mediated platform work.

3. Survey Screening (Blocks 5.1/5.2)

3

Target Groups & Screening Questions

Group A: Self-Employed (11, 12, 21)

Col (15) Block 5.1 / Col (14) Block 5.2: "Provided goods/services by accessing them through a digital platform?"

Group B: Employees/Casual (31, 51)

Col (16) Block 5.1 / Col (15) Block 5.2: "Provided goods/services while working for a digital platform?"

The Major Mode Test (Col 17/16)

  • Enter Code 1 (Yes) if platform work was the majority mode (majority of time) during the reference period.
  • Otherwise, enter Code 2.

4. Practical Case Studies

4

Classification Logic

  • Status 31 (Regular Employee): Code as platform worker only if the employer-employee relationship is difficult to justify in the traditional sense.
  • Casual Wage Labour (Status 51): If an app-driver hires someone else on daily wages to drive their cab (due to sickness etc.), the hired person is a platform casual worker.
  • The "Shop" Rule: A shopkeeper selling via an app is a platform worker, but their physical shop employees are not.
NIC Exclusion: Platform work does not apply to Industry Section B (Mining and Quarrying).
